# H4
## Tehtävän kuvaus
Tässä tehtävässä mallinnetaan yksinkertainen auton rakenne C++:ssa käyttäen **kompositiota**. Tehtävässä luodaan seuraavat luokat:  

- `Engine` – moottorin tiedot  
- `Wheel` – renkaiden tiedot  
- `Car` – koostuu yhdestä `Engine`-oliosta ja neljästä `Wheel`-oliosta  

Auton tiedot tulostetaan konsoliin.
